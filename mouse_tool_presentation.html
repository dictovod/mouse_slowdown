Mouse Speed Tool

Регулировка скорости мыши клавишами громкости — без меню, без лишних движений.

▶ powershell -ExecutionPolicy Bypass -File mouse_tool.ps1
Принцип работы

Как это устроено

1

Перехват нажатий через Raw Input API

Программа регистрирует скрытое окно и подписывается на сырые события клавиатуры (Usage Page 0x01/0x06) и Consumer Control (0x0C/0x01) — напрямую от драйвера, без захвата фокуса.

2

Двойное нажатие активирует режим

Однократное нажатие Volume Up/Down — только «первый тап». Два нажатия подряд (менее 500 мс) активируют режим регулировки на 2 секунды.

3

Изменение скорости через SystemParametersInfo

В активном режиме каждое нажатие вызывает SPI_SETMOUSESPEED с сохранением в реестр (SPIF_UPDATEINIFILE). Диапазон: 1–20.

4

Атомарный таймер без потоков

Два глобальных AtomicU64 хранят метки времени последнего нажатия и конец активного окна — race-free, без мьютексов.

Возможности

Ключевые особенности

🎯

Без фокуса окна

Raw Input API работает даже когда активно другое приложение — игра, браузер, редактор.

⚡

Нативная скорость

Компилируется в .exe без рантайма. Запуск мгновенный, потребление RAM менее 2 МБ.

🛡️

Защита от случайных нажатий

Двойное нажатие за 500 мс — активация. Одно нажатие громкости ничего не меняет.

💾

Сохраняется в реестр

Скорость записывается через SPIF_UPDATEINIFILE — настройка сохраняется после перезагрузки.

🔌

Consumer Control

Поддержка HID Usage Page 0x0C — медиаклавиши на клавиатурах и гарнитурах.

📦

Один .ps1 — всё

Скрипт сам ставит Rust, MSVC Build Tools, компилирует и запускает. Нужен только PowerShell.

Установка

Запуск за три шага

01

Скачай скрипт mouse_tool.ps1

Invoke-WebRequest -Uri "https://raw.githubusercontent.com/dictovod/mouse_slowdown/main/mouse_tool.ps1" -OutFile mouse_tool.ps1
02

Запусти в PowerShell (Rust и Build Tools поставятся автоматически)

powershell -ExecutionPolicy Bypass -File mouse_tool.ps1
03

Нажми Volume Down/Up дважды быстро для активации режима регулировки. Дальше нажимай по одному разу.
